What is the VRR Toolkit?

Making dungeons [for tabletop games like D&D] is a pain. Let’s make it easier! While we’re at it, we can also make it so people can traverse them using a VR headset or mobile device. Goals:

  • Allow a user to use 2D tools to generate the layout of a dungeon, floor by floor, including rooms, passageways, doors, traps, and props.
  • Generate 2D dungeons randomly, using soft-inputs defined by the user.
  • Take this 2D information, and generate a 3D environment from it that can be traveled through using VR (mobile first). This travel will be on “rails.”
  • Print the 2D dungeon representation as a map users can utilize to play D&D.
  • Allow user to control their movement in the VR environment using a controller, rather than being on rails.

My Role - Associate Producer
