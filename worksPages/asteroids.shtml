Ebola: Endemic (Prototype)


The Project

Built using Processing 2.2.1.

Ebola: Endemic is a quick and dirty rework of the retro game Asteroids. The desire was to make a prototype of a game with the same mechanics as Asteroids, with a bit of a goofy back story.

I used my experience from the Chasing and Fleeing prototype to create a background scene for the main menu. Overall it was entertaining to create, and I feel it was a valuable experience utilizing vectors and keyboard input.





My Responsibilities - Programmer

Programmer
Main menu is animated with "infected cells" floating in the background. Utilized user input to steer the boat, fire bullets, and navigate menus. All targets are created in random locations with initial vectors toward an area surrounding the player. The player may accelerate to a maximum velocity, and the ship is affected by friction, eventually coming to a stop. All objects in the game wrap around the screen, similar to the original Asteroids. When larger asteroids are destroyed, it spawns 3 smaller ones that start with their own random direction and a constant velocity.
