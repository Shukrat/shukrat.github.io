What is the John Given Wines (Android app)?

This application is the first step in a comprehensive update to John Given Wines, Inc. online presence. The app displays information about the company, upcoming tastings, new vintages, current wines, and the regions they're from.

The app has been coded in a way that allows for people with very little coding knowledge to update the content easily. It checks with a file on the John Given Wines website to see if any updates have occured, and if so, updates the content in the app. All content is generated through the use of JSON files, making for easy editting.

My Role - Android Developer, UI Designer

I started this project as a way to learn Android development. I did the code architecture, refactoring (as necessary), put together the JSON files, and did all of the coding and debugging. Overall, it's been a very enjoyable experience, and I've found that Android seems to be held together through magic and duct tape.

Currently working on recuding the minSDK from 21 to 17 through UI redesign. Will also be adding a search function to the "All Wines" section. Additionally, I will be updating the JGWines website.
